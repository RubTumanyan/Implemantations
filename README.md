 My STL Implementations (C++)

This repository contains my custom implementations of core C++ Standard Template Library (STL) components for learning purposes. The goal of this project is to deeply understand how STL works internally, including memory management, allocators, move semantics, and data structure design.

 What’s Inside

This project includes simplified but functional versions of:


 Custom Allocator — basic implementation using ::operator new / ::operator delete
 Memory management system using allocate, construct, destroy, deallocate
 Copy and Move semantics (Rule of Five)
 Dynamic resizing and capacity management
